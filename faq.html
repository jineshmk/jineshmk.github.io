Frequently Asked Questions

How this application works?

This application works in response to the user input. User can select the any locality (Panchayat/Municipality/Corporation) in Trivandrum district from the dropdown list provided and the application can predict the Dengue fever risk factor in the select area based on a mathematical model works behind the application

Will the application use for any locality?
Currently this application works in a boundary of aPanchayat/Municipality/Corporation. All the calculations are based on a defined area and using the survey details and documented dengue fever incidents for the previous years. We are planning to implement independent location based prediction in the next update of the project.

How can I use this application on my mobile?
This application can be installed very easily in your mobile phone as same as any other application you install in your phone.

What mathematical model work with this application?
This application uses a SEIR (Susceptible, Exposed, Infected and Recovered) model behind. SEIRmodel is a an accurate model to predict the epidemic caused by infectious diseases. This mathematical model was build based on the Dengue fever occurrences and climatic factors implemented in a host vector compartmental model.
